# @agent-society/core

Core library for Nostr-based ILP peer discovery and SPSP.

## Install

```bash
npm install @agent-society/core
```

## Usage

```ts
import {
  NostrPeerDiscovery,
  NostrSpspClient,
  NostrSpspServer,
  SocialTrustManager,
  BootstrapService,
} from '@agent-society/core';
```
